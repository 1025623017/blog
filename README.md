## [APICloud](http://www.apicloud.com/)开发小结

- [教程](http://community.apicloud.com/bbs/forum.php?mod=viewthread&tid=9306)

- [工具](http://www.apicloud.com/devtools)
  - 推荐：APICloud Studio - Eclipse Plugins

- 推荐的目录结构
  - css[] 
  - error[] 
    - error.html `404页面`
  - feature[] `存放模块资源`
  - html[] 
  - icon[] `APP图标`
  - image[] 
  - launch[] `APP启动页`
  - res[] `存放音频视频、加密信息等资源`
  - script[] 
  - wgt[] `存放子widget`
  - config.xml `配置文件`
  - index.html
